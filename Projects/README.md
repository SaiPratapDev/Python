# Server Health Monitoring (Python)

A Python automation project to monitor server health on Ubuntu servers.

## Features
- CPU usage monitoring
- RAM usage monitoring
- Disk usage monitoring
- GPU usage monitoring (if available)
- Hourly automation using cron jobs
- CSV-based historical logging

## Tech Stack
- Python
- Linux (Ubuntu)
- psutil
- Cron jobs

## Use Case
Useful for deployment and operations teams to proactively monitor server health and troubleshoot issues.
